# Product Images Directory

Add the following images to this folder with the exact filenames:

## Required Images

### Logo
- `logo.png` - Company logo (48x48px or larger)

### Hero Section
- `hero-background.webp` - Hero banner background image (1920x600px recommended)

### Product Images
- `rubber-bushes.webp` - Rubber Bushes product image
- `rubber-gaskets.webp` - Rubber Gaskets product image
- `o-rings.webp` - O Rings product image
- `rubber-seals.webp` - Rubber Seals product image
- `rubber-bellows.webp` - Rubber Bellows product image
- `anti-vibration-mounts.webp` - Anti Vibration Mounts product image
- `rubber-pads.webp` - Rubber Pads product image
- `custom-molded-parts.webp` - Custom Molded Rubber Parts product image

## Image Specifications

- **Format**: WebP for optimal compression and performance
- **Product Images**: 600x400px minimum (3:2 aspect ratio)
- **Logo**: Square PNG, at least 48x48px
- **Hero Background**: 1920x600px or 1920x800px

## Notes

- Use WebP format for all product and background images for better performance
- Ensure all images are optimized for web to maintain fast loading
- Product images should have white or neutral background for consistency
